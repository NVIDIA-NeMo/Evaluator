## Deployment Types

Choose the deployment type for your evaluation:

<Cards>

<Card title="None (External)" href="/libraries/nemo-evaluator-launcher/configuration/deployment/none">

Use existing API endpoints. No model deployment needed.
</Card>

<Card title="vLLM" href="/libraries/nemo-evaluator-launcher/configuration/deployment/vllm">

Deploy models using the vLLM serving framework.
</Card>

<Card title="SGLang" href="/libraries/nemo-evaluator-launcher/configuration/deployment/sglang">

Deploy models using the SGLang serving framework.
</Card>

<Card title="NIM" href="/libraries/nemo-evaluator-launcher/configuration/deployment/nim">

Deploy models using NVIDIA Inference Microservices.
</Card>

<Card title="TRT-LLM" href="/libraries/nemo-evaluator-launcher/configuration/deployment/trtllm">

Deploy models using NVIDIA TensorRT LLM.
</Card>

<Card title="Generic" href="/libraries/nemo-evaluator-launcher/configuration/deployment/generic">

Deploy models using a fully custom setup.
</Card>

</Cards>

## Quick Reference

```yaml
deployment:
  type: vllm  # or sglang, nim, none
  # ... deployment-specific settings
```
