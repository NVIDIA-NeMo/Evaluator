# Issue tracker: Linear

Issues and PRDs for this repo live in Linear.

## Conventions

- Team: Frontier AI Evaluation
- Ticket prefix: `EVAL-`
- Use the Linear connector/plugin to search, create, update, and link issues.
- Use the repo's GitLab merge requests for code review and implementation links.

## When a skill says "publish to the issue tracker"

Create a Linear issue on the Frontier AI Evaluation team.

## When a skill says "fetch the relevant ticket"

Fetch the Linear issue by its `EVAL-` identifier.
