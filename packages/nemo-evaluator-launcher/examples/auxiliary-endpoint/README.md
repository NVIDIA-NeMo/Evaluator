# Auxiliary Endpoint Examples

primary model deployment. For full documentation, see the
[Auxiliary Deployments guide](../../../../docs/libraries/nemo-evaluator-launcher/configuration/auxiliary-deployments.md).

## Examples

| File | Description |
|---|---|
| `slurm_vllm_judge.yaml` | Deploy a vLLM judge model alongside the primary model for AIME LLM-as-a-judge evaluation on Slurm. |
